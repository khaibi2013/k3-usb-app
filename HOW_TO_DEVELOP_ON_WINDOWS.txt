Hướng Dẫn Phát Triển Ứng Dụng IronVault Trên Windows
======================================================

Để tiếp tục code ứng dụng này trên môi trường Windows vào ngày mai sau khi bạn tải (pull) code từ Git xuống, hãy làm theo các bước đơn giản sau:

### Bước 1: Tải Code Mới Nhất Từ Git
Mở ứng dụng Command Prompt (CMD), PowerShell hoặc Git Bash, di chuyển tới thư mục bạn muốn lưu dự án (ví dụ: ổ D:) và chạy lệnh:
> git clone <đường_dẫn_git_của_bạn_nếu_chưa_clone>
> cd k3-usb-app

(Nếu bạn đã clone dự án từ trước, chỉ cần vào thư mục `k3-usb-app` và chạy lệnh: `git pull` để cập nhật code mới nhất vừa được chỉnh sửa trên Mac).

### Bước 2: Cài Đặt Các Gói Thư Viện (Dependencies)
Cài đặt Node.js (phiên bản 18 hoặc 20 trở lên). 
Trong thư mục `k3-usb-app`, hãy chạy lệnh sau để tải tất cả thư viện cần thiết:
> npm install

### Bước 3: Chạy Môi Trường Lập Trình (Dev Mode)
Sau khi tải xong thư viện, bạn dùng lệnh sau để vừa bật giao diện lập trình, vừa chạy Electron:
> npm run dev

Code trên Windows sẽ tự động hot-reload y hệt như trên Mac. Mọi tính năng bảo mật, mã hóa, tìm kiếm (cả dấu `\` của Windows) đều đã được tối ưu hóa để chạy xuyên nền tảng.

### Bước 4: Đóng Gói (Build) Ra File .EXE Chạy Trực Tiếp Trên USB
Khi bạn code xong và muốn xuất ra một file chạy `.exe` dạng Portable (Không cần cài đặt, copy thẳng vào USB là chạy), hãy chạy lệnh:
> npm run pack:win

Sau khi lệnh chạy xong 100%, hãy vào thư mục `release/` bên trong dự án. Bạn sẽ thấy một file tên là `IronVault_Windows.exe`.
Hãy copy file `.exe` này vào thẳng USB (cùng với thư mục `.vault` nếu có) là đã có thể chạy trực tiếp IronVault từ USB trên mọi máy Windows!

---
### 🤖 BÍ KÍP DÙNG ANTIGRAVITY IDE TRÊN WINDOWS (NGÀY MAI)
Vì Windows đôi khi bắt xác thực Token Git rất phiền phức, bạn hoàn toàn có thể BỎ QUA Git!
1. Copy file `IronVault_SourceCode.zip` (trong thư mục `release/` của Mac hiện tại) vào USB.
2. Sang máy Windows, copy file Zip đó ra máy và giải nén thành một thư mục.
3. Mở Antigravity IDE trên Windows, chọn thư mục vừa giải nén đó, và copy-paste **nguyên văn câu lệnh dưới đây** gửi cho AI:

*"Chào AI, đây là mã nguồn dự án Két sắt mà chúng ta đã làm trên Mac. Hãy tự động chạy lệnh `npm install` để cài đặt môi trường. Sau đó, bắt buộc phải đọc thật kỹ file `AI_CONTEXT.md` có sẵn trong thư mục này để hiểu toàn bộ ngữ cảnh dự án, các lỗi Cross-platform đã xử lý hôm qua trước khi chúng ta bắt đầu code tiếp."*

AI sẽ tự động cài đặt mọi thứ và hiểu ngay lập tức!

Chúc bạn lập trình vui vẻ!
